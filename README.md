# 📚 AI Study Assistant Pro

An AI-powered learning platform that enables students to interact with study materials using Retrieval-Augmented Generation (RAG), LangGraph, and Groq LLMs.

## 🚀 Features

- 📄 Multi-PDF Upload & Processing
- 🤖 AI-Powered Question Answering
- 🔍 Retrieval-Augmented Generation (RAG)
- 🎤 Mock Interview Generator
- 📝 AI Answer Evaluation
- 📊 Learning Progress Dashboard
- 🔐 User Authentication
- ☁️ MongoDB Integration
- 🧠 LangGraph Workflow Support

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- LangGraph
- FAISS
- Sentence Transformers
- MongoDB

## ⚙️ Installation

```bash
git clone https://github.com/kushant451/AI-Study-Assistant-Pro.git
cd AI-Study-Assistant-Pro

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
```

## 📌 How It Works

1. Upload PDF study material
2. Process documents into embeddings
3. Ask questions from PDFs
4. Generate mock interview questions
5. Receive AI-powered feedback
6. Track learning progress
